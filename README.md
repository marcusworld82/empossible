# Empossible

AI Creative Agency website — built with plain HTML, CSS, and JavaScript.

## Pages
- `index.html` — Homepage
- `services.html` — Services
- `portfolio.html` — Portfolio
- `pricing.html` — Pricing
- `about.html` — About
- `book.html` — Book a Call

## Deploy to Vercel
1. Push this repo to GitHub
2. Log in to [vercel.com](https://vercel.com)
3. Click "Add New Project"
4. Import your GitHub repo
5. Vercel auto-detects static HTML — click Deploy
6. Done. Your site is live.

## Adding Calendly to Book Page
1. Go to [calendly.com](https://calendly.com) and create a free account
2. Create a 20-minute event type
3. Click "Add to Website" and copy the inline embed code
4. Open `book.html` and replace the `calendly-placeholder` div with the Calendly embed code
